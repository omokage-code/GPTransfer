# Privacy Policy

Last updated: 2026-06-25

GPTransfer is a local macOS app for transferring files from a connected GoPro camera to a folder selected by the user.

## Data Collection

GPTransfer does not collect analytics, user accounts, personal information, or payment information.

## File Transfers

Camera files are transferred locally from the connected camera to the selected folder on the Mac.

GPTransfer does not upload camera files to an external server.

## Camera Access

GPTransfer reads media information from the connected camera in order to show file lists, thumbnails, sizes, dates, and transfer status.

Camera files are treated as read only. GPTransfer does not delete or modify files on the camera.

## Local Files

During transfer, GPTransfer may create `.partial` files in the selected save folder. These files are used to avoid treating interrupted transfers as completed files.

If a transfer is canceled or interrupted, unfinished `.partial` files may remain in the selected save folder until the user reviews or removes them.

## Network Access

GPTransfer connects to the camera through the local camera media interface. This may use a USB network connection or a local network connection, depending on the camera and Mac configuration.

GPTransfer does not use this connection to send files to any external service.

## Support Links

GPTransfer may include links to external support pages, such as Buy Me a Coffee. Opening those links is optional and handled by the user's browser.

External services have their own privacy policies.

## Contact

For project support, use the support links or repository contact information provided with the public release.
